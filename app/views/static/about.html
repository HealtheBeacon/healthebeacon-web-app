HEALTHEBEACON | ABOUT

  With the intent of creating healthier teams and lifestyles, the HealtheBeacon application was created for the 2016 Cerner HackFest. Utilizing an iOS application, the HealtheBeacon app uses Bluetooth Low Energy (BLE) to connect to a Rasberry Pi. One this information is trasnferred to the Raspberry Pi, this data is ingested into a Ruby on Rails web application which manipulates the data, runs an algorithm to caculate the score and serves it all up in an easy to read leaderboard.

The technologies utilized include the following:

  • iPhone Application Development
  • Raspberry Pi
  • Bluetooth Low Energy
  • SQL Lite
  • Ruby on Rails
  • Bootstrap
  • html
  • CSS
  • Adobe Illustrator
  • Trello

These are just a few of the languages and technologies utilized throughout the past week in the creation of the project.

Created By: Caleb Bertels, Jack Considine, Nate Demery and Andrew McKittrick.
